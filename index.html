Leaflet.shim

A shim layer to disable interactive vector features and markers without having to hide or update them. It basically propagates all the mouse events straight to the map, bypassing the layers.

Leaflet shim(backdrop)

Particularly useful to accompany controls like Leaflet.Measure, while having some features on the map that will respond to clicks or won't propagate events essential for the drawing process.


    var s = new L.Shim().addTo(map);
